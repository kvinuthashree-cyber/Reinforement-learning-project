
Q-Learning Maze Game
=====================

This is a simple reinforcement learning project using Q-learning to solve a maze.

How to Run
----------
1. Install dependencies:
   pip install numpy pygame

2. Run the game:
   python maze_game.py

Controls
--------
- M = Toggle Manual / AI mode
- Arrow Keys = Move manually (in manual mode)
- R = Restart
- Close window to quit

Files
-----
- maze_env.py: Maze environment with obstacles
- q_learning_agent.py: Q-learning logic
- maze_game.py: Game with visualization using Pygame
